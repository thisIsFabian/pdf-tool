Known Limitations

When combining multiple PDF files the internal links (to different pages of the PDF) of the appended files will break.

Privacy

Your files will never leave your device and you have unlimited usage of this tool for free since I don't have to run a server. After the page loaded you can disable your internet connection and everything will still work. Additionally this project is open source.

Technology

This website uses an actively maintained fork of the pdf-lib library for handling PDF files.
